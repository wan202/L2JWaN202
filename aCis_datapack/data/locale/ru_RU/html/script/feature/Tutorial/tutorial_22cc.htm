[Перевод Класса Человеческого Волшебника]:
Для того чтобы стать Чернокнижником, Волшебник должен успешно завершить три задания. Испытание Ученого доступно только игрокам с уровнем 35 и выше. Начало этого задания связано с встречей с Магистром Мириеном из Диона Касл Тауна. Свидетельство Доверия может быть получено только игроком с уровнем 37 и выше. Начало этого задания связано с встречей с Верховным Жрецом Холлинтом из Орена Касл Тауна. Наконец, Испытание Призывателя доступно игроку с уровнем 39 и выше. Начало этого задания связано с встречей с Верхним Призывателем Галатеей из Деревни Глудин.
